02

Allgemeines zur Datenverarbeitung

Die App dient der Erstellung von Lerninhalten auf Grundlage von durch Nutzer und Nutzerinnen bereitgestellten Texten.

Ein Teil der Funktionen ist kostenpflichtig und kann über ein In-App-Abonnement freigeschaltet werden.

Die Verarbeitung personenbezogener Daten erfolgt ausschließlich in dem Umfang, der zur technischen Bereitstellung der App-Funktionen erforderlich ist.

Eine Registrierung oder Erstellung eines Accounts ist nicht vorgesehen.

03

Diese Website

Diese Website ist die Begleitseite zur iOS-App Encognition. Sie dient ausschließlich der Information über die App und enthält keine interaktiven Funktionen, die eine Datenerhebung erfordern würden.

Cookies und Tracking: Diese Website verwendet keine Cookies, keine Tracking-Pixel und keine Analyse- oder Werbedienste. Es werden keine Nutzungsprofile erstellt.

Hosting: Die Website wird über Cloudflare Pages bereitgestellt. Beim Aufruf der Seite werden dabei technisch bedingt Verbindungsdaten verarbeitet — insbesondere IP-Adresse, Zeitstempel, aufgerufene URL und Browser-Informationen. Diese Verarbeitung liegt im Verantwortungsbereich von Cloudflare und ist für die technische Auslieferung der Website erforderlich: cloudflare.com/privacypolicy

Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse am Betrieb einer funktionsfähigen Website).

04

Datenverarbeitung auf dem Endgerät

Folgende Daten werden lokal auf dem Endgerät gespeichert und verarbeitet:

  • hochgeladene Dokumente
  • extrahierte Textinhalte
  • generierte Lerninhalte
  • lokale Bibliothek
  • App-Einstellungen

Diese Daten verbleiben grundsätzlich auf dem Endgerät, sofern keine servergestützte Funktion genutzt wird.

Bei der ersten Nutzung wird zusätzlich eine zufällig generierte Installations-ID erstellt und lokal gespeichert. Diese wird bei Nutzung servergestützter Funktionen an den Server übermittelt und dient ausschließlich:

  • der technischen Zuordnung von Anfragen
  • der Missbrauchsvermeidung
  • der Validierung von Abonnements

Die Installations-ID enthält keine unmittelbar identifizierenden Angaben wie Name, E-Mail-Adresse oder Zahlungsdaten.

05

Übermittlung von Daten an Server

Bei Nutzung servergestützter Funktionen werden folgende Daten verarbeitet:

a) Zur Erstellung von Lerninhalten

  • Textauszüge aus bereitgestelltem Material
  • Installations-ID
  • Dokument-Hashwerte
  • technische Metadaten (z. B. Zeitstempel, Anfrageparameter)

b) Zur Validierung von Abonnements

  • Produktkennung (Product ID)
  • Transaktionskennung
  • Ablaufdatum
  • Abonnementstatus
  • Validierungsnachweis (Receipt / Token)

Die Datenübertragung erfolgt verschlüsselt (TLS).

Es werden keine Zahlungsdaten wie Kreditkarten- oder Bankinformationen verarbeitet oder gespeichert. Zur Freischaltung kostenpflichtiger Funktionen wird lediglich der Abonnementstatus verarbeitet.

Eine Profilbildung oder weitergehende Auswertung von Kaufhistorien erfolgt nicht.

Für die Zahlungsabwicklung gelten ergänzend die Datenschutzbestimmungen von Apple:
apple.com/legal/privacy
